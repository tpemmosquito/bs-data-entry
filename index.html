Safe Labguru Export

This workflow never overwrites prior data. Every upload is a new versioned CSV or an updates-only CSV. API keys are not saved.

Security rule: enter the Labguru API key only when uploading. The key is kept in memory for the request only and cleared immediately after. For production, use a secure HTTPS backend upload endpoint instead of calling Labguru directly from this file.
